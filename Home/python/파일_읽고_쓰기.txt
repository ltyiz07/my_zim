====== 파일 읽고 쓰기 ======
Created 수요일 29 7월 2020

기본적으로 텍스트 형식으로 저장.
'w' : 쓰기
'r' : 읽기
'a' : 추가하기

"wb" : 바이너리로 쓰기
"...
...


1. 파일을 읽고 판매건수를 가져오기
2. 읽어 온 데이터를 2차원 리스트로 만든다
3. 동일 상품의 총판매수량을 계산
4. 계산결과를 csv로 저장해서 구글 스프레스시트에서 읽는다
