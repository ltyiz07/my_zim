====== 함수 def ======
Created 화요일 28 7월 2020

기존의 변수는 globla frame 에 저장된다.

 그러나 함수는 별도의 Frame 을 만들고 거기에서 작동한다. 그러므로 함수내에서의 변수들은 함수 내에서만 사용이 된다. : 이런것들을 함수는 자기만의 스코프(범위)를 갖는다 라고한다. (외부에서는 함수내부의 변수를 사용할수 없다 그러나 함수 내부에서 외부의 변수는 사용 가능하다.)
 **단 외부의 변수를 사용하고 싶다면 global 명령어를 사용하면 된다.**(그러나 가급적이면 함수안에서는 함수 내부의 변수만을 사용하는 것이 권장됨으로 사용하지 않는것이 좋다.)

 또한 함수는 입출력이 없는 함수, 입력과 출력 둘중 하나만 있는 함수, 입출력 모두 있는 함수 가있다. 그런데 출력은 항상 하나이다.

함수의 이름은 소문자로 시작한다. 주로 스네이크케이스를 사용한다.(코딩 컨벤션)
