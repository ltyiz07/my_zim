====== cmd ======
Created 화요일 21 7월 2020

cd / : 루트 디렉토리로 이동
dir : 폴더 목록을 보여줌
dir [[/s]] : 폴더에 서브 폴더를 포함한 목록을 보여준다

절대경로
해당 드라이브의 로트로부터 경로를 지정하여 작업을 진행 
루트는 윈도우에서는\ 이고 리눅스에서는/ 이다.
(같은 드라이브에서 작업하는 경우 드라이브명을 생략하고 루트로부터 절대경로를 지정할 수 있다.)역슬레쉬


같은 드라이브에 있을 때에는 드라이브는 생략 가능
