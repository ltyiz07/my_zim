====== git ======
Created 화요일 21 7월 2020


[[+명령어들]]


**"commit**"은 일종의 스텝을 정하는 것이다.
수없이 많은 작업을 하는데 그 모든것을 모두 기록할 수 없다 그래서 내가 커밋한 상태만 보관이 되는 것이다.
인덱스에 추가된 파일들이 깃 관리가 된다.
https://github.com/ltyiz07/my_repo 읽어보기

git diff <원래가지> <비교대상 가지>

git remote add origin https://github.com/ltyiz07/my_repo.git
git push origin master	#push
git clone https://github.com/poby13/my-repo.git	#깃 클로닝, 내려받기
